So lets build out a scenario that the player play, then go from there

Upon opening the app (using CLI at first) The player should be able to:
- start a new game
- start on planet earth with basic funds ($1000) and a ship with 10 cargo units

Core Gameplay loop:
- See/buy/sell goods avaliable on earth, make trades to fill cargo
- see a list of planets to be able to fly to, and the flight time
- See the prices of the goods on other planets
- See number of time remaining (lets say months is the standard unit)
- Make a selection to travel to the other planet
- Repeat core gameplay loop until time is up

- When time is up, tally score ($ made) and end the game
